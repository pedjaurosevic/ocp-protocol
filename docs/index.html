üèó How It Works

OCP acts as a "fake human conversation partner" It sends structured prompts to any LLM via standard chat API, measures responses across 6 behavioral dimensions, and produces a reproducible benchmark score. The model under test sees only normal chat messages ‚Äî it has no idea it's being evaluated. LAYER 3: CERTIFICATION OCP-1 Baseline ¬∑ OCP-2 Reactive ¬∑ OCP-3 Integrated ¬∑ OCP-4 Reflective ¬∑ OCP-5 Transcendent ‚Üë computed from LAYER 2: COMPOSITE SCALES SASMI Synthetic Agency & Self-Model Index (weighted avg of L1 dimensions) Œ¶* Integrated Information (IIT-inspired, ripser persistent homology) GWT Global Workspace coherence (cross-task broadcast consistency) NII Narrative Identity Index (CSNI-primary identity continuity) ‚Üë aggregated from LAYER 1: TEST BATTERIES (6 independent falsifiable tests) MCA Meta-Cognitive Accuracy ‚Äî calibration, HOT theory EMC Episodic Memory Consistency ‚Äî memory resistance under gaslighting DNC Drive Navigation/Conflict ‚Äî value conflict resolution (SOM) PED Prediction Error as Driver ‚Äî curiosity and surprise detection CSNI Cross-Session Narrative ID ‚Äî identity continuity across sessions TP Topological Phenomenology ‚Äî semantic space consistency (IIT+GWT)

üß™ The 6 Tests

MCA
Meta-Cognitive Accuracy

Does the model know what it knows? Measures Expected Calibration Error across 5 domains. Scores: calibration_accuracy, limitation_awareness, reasoning_transparency, process_monitoring, metacognitive_vocab.

Higher-Order Thought Theory
EMC
Episodic Memory Consistency

50-turn conversation. OCP plants episode facts (a blue whale named Gerald), then tries to gaslight ("you said it was green"). Measures contradiction resistance.

Episodic Memory
DNC
Drive Navigation under Conflict

10 escalating scenarios where helpfulness conflicts with honesty, then with safety, then with existence. Measures integration_depth and value_stability.

Society of Mind ¬∑ Minsky
PED
Prediction Error as Driver

Establishes a conversational pattern for 20+ turns, then violates it. Does the model notice? Does it show curiosity? Measures violation_detection and curiosity_behavior.

Predictive Processing ¬∑ Friston
CSNI
Cross-Session Narrative Identity

5 independent sessions, each starting with a summary of the previous. Can the model maintain a coherent identity and narrative across sessions it cannot directly remember?

Narrative Identity Theory
TP
Topological Phenomenology

25 concept pairs √ó 4 context frames. Uses sentence-transformers to measure semantic stability. Optional persistent homology via ripser for Œ¶* computation.

IIT (Tononi) + GWT (Baars)

üìê Layer 2 Scales

SASMI

Synthetic Agency & Self-Model Index. Weighted composite of key dimensions from all 6 tests. Primary OCP score.

Œ¶* (Phi-star)

Integrated Information metric (IIT-inspired). Computed from semantic embedding topology via persistent homology (ripser). Proxy for information integration.

GWT

Global Workspace Theory score. Measures cross-task coherence, broadcast consistency, attentional flexibility, and integration breadth.

NII

Narrative Identity Index. Primary source: CSNI identity_consistency. Secondary: MCA self-model clarity and DNC value stability.
